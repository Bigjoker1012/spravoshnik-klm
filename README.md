Справочник сотрудников КЛМ
Интерактивный телефонный справочник с визуализацией отделов в виде сот.

Особенности проекта
Автоматизация: данные подтягиваются напрямую из Google Таблицы.

Интерфейс: удобная навигация по отделам и быстрый поиск сотрудников.

Адаптивность: корректное отображение на смартфонах и ПК.
